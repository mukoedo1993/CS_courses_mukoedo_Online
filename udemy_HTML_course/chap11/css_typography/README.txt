CSS typography:
It is the art of arranging type or text so that it is easy to read
and support the overall design of a project.


@font-face:
To use any fonts you have license to use.


Tips on font-size:
You should define a baseline size on either your body or each HTML element
so that you can trickle down to every element on your page.


font-weight:
To manage the boldness of the text.

font-style:
e.g.: italic


letter-spacing:
space between each letter.

word-spacing:
space between each word (on the basis of letter-spacing).
