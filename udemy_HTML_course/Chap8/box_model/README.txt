A web page is a collection of boxes.
Web designer:
learn to manipulate and decorate boxes in any way that
you can imagine.


A box is nothing more than an area that takes up space on
the web page 

Padding:
to create space within the inside of that box.
The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element
 (top, right, bottom, and left).




Border: to draw an outline around the outer
edge of that box.

Margin: then to push or create space around the outside of the
box, we use margin.
