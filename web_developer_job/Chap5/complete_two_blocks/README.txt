Sidenote:
rem: Whenever we declare a font size or a padding or margin value, we want to do
so in terms of rem instead of pixels or percentages or em.
Everything is relative to the root of the page, i.e., html element.
Most Web browsers will have a default font size on the HTML element of about the
equivalent of 16 pixels.
Not everyone in the world have the configure of browser exactly same.
We want to honor the users' font size preferences.
